# 환경 설정

- java_jdk version으로 `1.8.0`을 필요로 한다.

`$ sudo apt-get update`

`$ sudo apt-get install openjdk-8-jdk`
> jdk1.8.0 설치

`$ sudo update-alternatives --config java`
> 여러 java 버전이 설치되어 있는 경우 확인하고 변경할 수 있다.
