# Spark 분산 환경

`$ mkdir ~/adhoc`

`$ cd ~/adhoc`

- 콘솔 환경에서 수행

`$ export PYSPARK_DRIVER_PYTHON=jupyter-console`

- Spark를 기동하고 MongoDB용의 패키지 추가(3.1.1 - Spark Version)

`$ pyspark --packages org.mongodb.spark:mongo-spark-connector_2.11:3.1.1`
